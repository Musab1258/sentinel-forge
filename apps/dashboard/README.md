# Dashboard

Operational dashboard for Sentinel Forge findings, scan history, trace evidence, and code-linked remediation views.

## Current role

The dashboard is the phase 4 visualization surface and remains part of the phase 6 demo path.

## What it demonstrates

- severity distribution
- detector concentration
- findings table and remediation context
- code-linked review flow

## Run locally

```bash
corepack pnpm --filter @sentinel-forge/dashboard dev
```
