# Contributing

Sentinel Forge is being built as a modular security platform for Soroban. Early contributions should bias toward clarity, reproducibility, and detector quality.

## Areas open for contribution

- static analysis rules and reporting
- vulnerable and secure Soroban contract examples
- architecture diagrams and threat models
- contributor tooling and developer workflow automation
- landing page and dashboard polish

## Standards

- keep Rust crates small and testable
- prefer deterministic fixtures for security tooling
- document new detectors with intent, severity, and false-positive tradeoffs
- include tests for frontend and engine changes when behavior changes
