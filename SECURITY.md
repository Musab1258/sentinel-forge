# Security Policy

Sentinel Forge handles security-sensitive tooling and research. Treat reports responsibly and avoid public disclosure of exploitable issues until maintainers have reviewed them.

## Reporting

- share impact, reproduction steps, and affected components
- include proof-of-concept material only when necessary
- avoid filing exploitable issues in public trackers until triaged

## Scope

- Rust engine crates
- example vulnerable contracts and fixtures
- landing page and dashboard code paths that expose user-provided inputs
