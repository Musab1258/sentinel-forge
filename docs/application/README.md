# Application Preparation

This directory collects the local, maintainership-facing assets that map to phase 6.

## Included materials

- `maintainer-brief.md`: short project positioning for reviewers and maintainers
- `demo-walkthrough.md`: reproducible demo flow across the analyzer, dashboard, and exploit lab
- `release-readiness.md`: local checklist for documentation, demos, and repo maturity

## Goal

Phase 6 is not about inventing a new subsystem. It is about making the current system legible, demoable, and credible enough for external evaluation.
